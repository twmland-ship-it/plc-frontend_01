# 電價實施設定功能新增

> **版本**: 2.2.28  
> **日期**: 2024-09-04  
> **類型**: 功能新增  
> **影響範圍**: 電費計算模組

## 📋 修改概述

在低壓非時間電價頁面的夏月期間設定右側，新增三個電價實施相關欄位，提供更完整的電價設定功能。

## 🎯 需求描述

根據使用者需求，在電費計算的低壓非時間電價頁面中，需要在夏月期間設定的右邊增加以下三個欄位：

1. **電價實施年** - 下拉式選單 (2020-2100)
2. **電價實施月** - 下拉式選單 (1-12)  
3. **臨時用電倍數** - 數字輸入框，預設值為1

#### 2. 新增響應式資料
```javascript
// 電價實施設定
const electricityRate = reactive({
  year: new Date().getFullYear(),      // 當前年份
  month: new Date().getMonth() + 1,    // 當前月份  
  temporaryMultiplier: 1               // 臨時用電倍數
})

// 年份選項 (2020-2100)
const yearOptions = computed(() => {
  const years = []
  for (let year = 2020; year <= 2100; year++) {
    years.push(year)
  }
  return years
})

// 月份選項 (1-12)
const monthOptions = computed(() => {
  const months = []
  for (let month = 1; month <= 12; month++) {
    months.push(month)
  }
  return months
})
```

### 輸入限制
- **年份**: 2020 ≤ year ≤ 2100
- **月份**: 1 ≤ month ≤ 12

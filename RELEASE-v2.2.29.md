## 🆕 主要新功能

### ⚡ 電費計算 - 低壓非時間電價年月選擇功能

#### 🎯 功能亮點
- **年月選擇介面**: 新增項目時先選擇電價實施年月 (2020-2100年, 1-12月)
- **臨時用電倍數**: 支援小數輸入，預設值為1，非臨時用電為1，臨時電依規定為大於1的小數
- **統一年月管理**: 移除元件內的年月選擇，改由主頁面統一管理
- **完整 API 整合**: 支援完整的電價設定 CRUD 操作

#### 🔧 技術改進
- **受控元件**: 低壓非時間電價元件改為受控元件，支援 v-model 雙向資料綁定
- **專用 Actions**: 新增專用 Vuex actions (`addLowVoltageNonTimeFee`, `editLowVoltageNonTimeFee`)
- **資料格式轉換**: 完整的 API 請求結構和資料映射
- **響應式設計**: 支援不同螢幕尺寸的年月選擇介面

#### 📋 API 更新
- `GET /electricity-rate/power-supply/0/segment/0/time-settings` - 獲取時間設定列表
- `POST /electricity-rate/power-supply/0/segment/0/details` - 新增/編輯電價設定
- 支援 TemporaryUsageMultiplier 臨時用電倍數參數
- 完整的 DetailItems 夏月/非夏月資料結構

## 🎯 使用者體驗

### 📱 操作流程
1. **選擇電價類型**: 系統設定 → 帳單管理 → 電費管理 → "低壓非時間"
2. **年月選擇**: 點擊「新增」→ 選擇年份和月份 → 確認
3. **詳細設定**: 進入電價設定頁面，標題顯示如 "新增2024-2"
4. **資料輸入**: 設定夏月/非夏月電價和臨時用電倍數
5. **儲存完成**: 點擊「儲存」完成設定

### 🎨 介面改進
- **清楚的標示**: 年月選擇 Modal 有明確的標題和說明
- **統一設計**: 與系統其他 Modal 保持一致的設計風格
- **響應式佈局**: 支援桌面和平板設備的操作
- **使用者友善**: 提供清楚的操作提示和錯誤處理

📊 低壓非時間電價
支援低壓非時間電價的完整設定，包含基本電費（裝置契約、需量契約）和流動電費的夏月/非夏月費率設定。
🌞 夏月期間設定
可自訂夏月期間的起始和結束日期，支援月份和日期的精確設定，預設為6月1日至9月30日。
📅 電價實施年月
支援2020年至2100年的電價實施年份設定，以及1-12月的月份選擇，確保電價政策的時效性。
⚡ 臨時用電倍數
支援臨時用電倍數設定，非臨時用電為1，臨時電依規定為大於1的小數，提供靈活的費率計算。

資料結構:

// 電價實施設定 const electricityRate = reactive({ year: new Date().getFullYear(), // 當前年份 month: new Date().getMonth() + 1, // 當前月份 temporaryMultiplier: 1 // 臨時用電倍數 }) // 年份選項 (2020-2100) const yearOptions = computed(() => { const years = [] for (let year = 2020; year <= 2100; year++) { years.push(year) } return years }) // 月份選項 (1-12) const monthOptions = computed(() => { const months = [] for (let month = 1; month <= 12; month++) { months.push(month) } return months })

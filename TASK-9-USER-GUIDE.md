# Task 9 使用指南：精確對齊功能

## 功能概述

精確對齊功能讓您可以精確控制 iframe 在頁面中的位置，包括：
- 獨立調整上、下、左、右邊距
- 一鍵水平置中
- 一鍵垂直置中
- 一鍵完全置中

## 如何使用

### 1. 啟用精確對齊模式

1. 打開「頁面設定」
2. 編輯或新增一個外部連結類型的頁面
3. 找到「精確對齊」區域
4. 勾選「啟用精確對齊模式」複選框

### 2. 使用快速置中按鈕

啟用精確對齊後，您會看到三個快速置中按鈕：

#### 水平置中 ⬌
- 點擊此按鈕自動計算並設定左右邊距
- iframe 會在水平方向上居中顯示
- 左右邊距會自動設定為相同的值

#### 垂直置中 ⬍
- 點擊此按鈕自動計算並設定上下邊距
- iframe 會在垂直方向上居中顯示
- 上下邊距會自動設定為相同的值

#### 完全置中 ⊕ (推薦)
- 點擊此按鈕同時進行水平和垂直置中
- iframe 會在頁面正中央顯示
- 所有四個邊距都會自動計算並設定

### 3. 手動調整邊距

如果您需要更精確的控制，可以手動調整每個邊距：

- **上邊距**：控制 iframe 距離頂部的距離
- **右邊距**：控制 iframe 距離右側的距離
- **下邊距**：控制 iframe 距離底部的距離
- **左邊距**：控制 iframe 距離左側的距離

每個邊距的範圍是 0-500px。

### 4. 即時預覽

調整邊距時，預覽區域會即時更新，讓您可以看到實際效果。

### 5. 儲存設定

確認設定無誤後，點擊「儲存」按鈕保存您的配置。

## 使用場景

### 場景 1：標準置中顯示
**目標**：讓 iframe 在頁面正中央顯示

**步驟**：
1. 啟用精確對齊模式
2. 點擊「完全置中」按鈕
3. 檢查預覽效果
4. 儲存設定

### 場景 2：靠左上角顯示
**目標**：讓 iframe 靠近左上角，但保留一些邊距

**步驟**：
1. 啟用精確對齊模式
2. 手動設定：
   - 上邊距：10px
   - 左邊距：10px
   - 右邊距：0px
   - 下邊距：0px
3. 檢查預覽效果
4. 儲存設定

### 場景 3：底部置中顯示
**目標**：讓 iframe 在底部水平置中

**步驟**：
1. 啟用精確對齊模式
2. 點擊「水平置中」按鈕
3. 手動調整：
   - 上邊距：設定為較大的值（例如 100px）
   - 下邊距：設定為較小的值（例如 10px）
4. 檢查預覽效果
5. 儲存設定

## 技術說明

### 邊距計算原理

**水平置中：**
```
左邊距 = 右邊距 = (可用寬度 - iframe 寬度) / 2
```

**垂直置中：**
```
上邊距 = 下邊距 = (可用高度 - iframe 高度) / 2
```

**完全置中：**
同時應用水平和垂直置中的計算。

### 配置存儲格式

邊距設定會以 JSON 格式存儲在資料庫中：

```json
{
  "margins": {
    "top": 10,
    "right": 20,
    "bottom": 10,
    "left": 20
  }
}
```

### 向後兼容性

- 舊的配置（沒有 margins 欄位）會自動使用預設值 0
- 不啟用精確對齊模式時，不會保存 margins 到配置
- 現有的顯示邏輯完全兼容

## 常見問題

### Q1: 為什麼點擊置中按鈕後邊距沒有變化？
**A:** 可能是因為 iframe 已經佔滿了可用空間。請檢查：
- 顯示模式是否設定為「等比例置中」
- 視窗大小是否足夠大
- 是否有其他元素佔用了空間

### Q2: 邊距設定後，Sidebar 收合時位置會改變嗎？
**A:** 是的，當 Sidebar 狀態改變時，可用空間會改變，iframe 會自動重新計算位置。如果您希望保持固定的邊距，建議在 Sidebar 收合狀態下設定邊距。

### Q3: 可以設定負數邊距嗎？
**A:** 不可以。邊距範圍限制為 0-500px，以確保 iframe 始終在可見區域內。

### Q4: 如何重置所有邊距？
**A:** 目前需要手動將所有邊距設定為 0。未來版本可能會添加「重置邊距」按鈕。

### Q5: 邊距會影響 iframe 的尺寸嗎？
**A:** 不會。邊距只影響 iframe 的位置，不影響其寬度和高度。iframe 的尺寸由「顯示模式」和「高度模式」決定。

## 最佳實踐

1. **先使用快速置中按鈕**：大多數情況下，「完全置中」按鈕就能滿足需求
2. **配合預覽使用**：調整邊距時隨時查看預覽效果
3. **考慮 Sidebar 狀態**：如果用戶經常切換 Sidebar，建議在收合狀態下設定邊距
4. **保持簡單**：除非有特殊需求，否則建議使用對稱的邊距（左右相等，上下相等）
